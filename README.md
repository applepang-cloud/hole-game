# 굴려라! — 데굴데굴 별왕자 🤴

반다이남코 「괴혼(카타마리 다마시)」 시리즈에서 영감을 받은 오리지널 로우폴리 3D 웹게임. Three.js 단일 HTML.

**플레이:** https://applepang-cloud.github.io/hole-game/

## 모드
- 🟢 **공 굴리기** — 끈끈한 공에 물건을 붙여 키운다
- ⚫ **구멍 빨아들이기** — 구멍으로 물건을 쏙쏙 삼킨다
- 🔵 **구멍 피하기** — 파란 볼로 작은 구멍은 없애고 큰 구멍은 피한다

## 조작
- 화면을 끌거나(터치/마우스) 방향키로 이동
- 📱 버튼: 기울기(자이로) 조작 — **모바일은 https 주소에서만 작동** (안드로이드 크롬은 비보안 origin에서 동작 센서를 막음)
